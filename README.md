# chat-app
chat-app is a Java-based client–server chat application that demonstrates core concepts of network programming
## Features
- TCP client–server communication
- Multi-threaded server
- Supports multiple clients
- Command-line client

## Technologies
- Java
- Sockets
- Multithreading
- Collections

## How to Run
1. Start the server  
2. Run one or more clients  
3. Exchange messages in real time
